Guess The Value Of a Box :

What To Do :

  • There is a square with 16 smaller squares inside.
  • Each smaller square contains a value between 1 to 21.
  • You can guess the value of any square by filling the input fields.
  • You can also reset the values by clicking the "Change Values" button.

Instructions:

  • You have to fill all the inputs first.
    • Fill the number of columns and rows between 1 and 4.
    • Guess the value between 1 and 21.
  • You cannot reveal the answer until you've tried at least once.
  • If you once revealed the answer, you have to change the values; otherwise, you cannot go ahead.
  • If you change the values once, you cannot reveal the answer until you've tried again (repeating).
